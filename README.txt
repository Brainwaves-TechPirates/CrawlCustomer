Crawl Customer: The idea of this application is to track and capture important actions and  click events performed on client side.

Normaly in webapplication we record information over http protocol into database or log events executed on server side into  log file, this application helps in recording important actions and click events performed on client side using a technique called "Click streaming".


This click streaming  will be used to perform data analytics to find out user experince on various bank services and product for  further value addation.


Technologies used:
Java,websockets,html5,glassfish (server),MongoDb,JEE71
